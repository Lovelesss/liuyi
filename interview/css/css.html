    # 1 属性选择器

    # 1.1 [attr]

    选择所有具有class属性的div元素

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>属性选择器</title>
    <style lang="stylus">
        div[class] {
            color: red
        }
    </style>
</head>
<body>
<div id="first">first</div>
<div class="second">second</div>
<div class="third">third</div>
</body>
</html>

    # 4.行内元素之间空隙的产生和去除

    之所以有空隙，是因为html文档里边两个span之间有很多空白字符。

    # 第一种方法：把空格去掉

    <div class="wrapper">
  <span class="box">111</span><span class="box">222</span>
</div>

    .wrapper {
